## 📌 Project Overview
This project presents an end-to-end **Customer Attrition Analysis** for a bank using **Python for data preprocessing** and **Power BI for interactive dashboards**.

The goal is to analyze customer behavior, identify churn patterns, and uncover key factors that influence customer retention.

---

## 🗂 Dataset Information
- **Source:** Kaggle
- **Rows:** 10,127
- **Columns:** 20+

### Key Features
- Customer_Age
- Gender
- Income_Category
- Card_Category
- Credit_Limit
- Total_Trans_Amt
- Total_Trans_Ct
- Months_Inactive_12_mon
- Contacts_Count_12_mon
- Avg_Utilization_Ratio
- Attrition_Flag (Target Variable)

---

## 🧹 Data Cleaning (Python)

The dataset was prepared using Python:

- Removed unnecessary columns
- Replaced "Unknown" values with mode for categorical data
- Detected and handled outliers
- Fixed data inconsistencies
- Prepared clean dataset for Power BI analysis
